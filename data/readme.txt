This modules contains:
1. [all_raw_pdf_files]: the raw pdf files downloaded from links mentioned below.
2. [separate_train_test.py]: the code to divide the raw pdf files into train & test folders

=================== TO run this module ==================
########## Navigate to this folder on CMD using below command:
cd data
########## Run below command
pip install -r requirements.txt
python separate_train_test.py

----------------------------------------------------------
Links used for downloading data:
https://stanfordnlp.github.io/pdf-struct-dataset/
https://cyberleninka.org/article/c/health-sciences
https://cyberleninka.org/article/c/mechanical-engineering

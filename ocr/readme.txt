This module contains:
1. [requirements.txt]: list of Python packages required to run this module
2. [read_pdf.py]: the code which extracts the text out of pdfs stored in train folder using OCR technology.

=================== TO run this module ==================
########## Navigate to this folder on CMD using below command:
cd ocr
########## Run below command
pip install -r requirements.txt
python read_pdf.py
